# Python GIF Generator

This is a mini project built using Python and the Pillow library.
It converts a sequence of images into an animated GIF.

# Technologies Used
- Python
- Pillow (PIL)
- OS module

How to Run
1. Install Pillow:
   pip install pillow
2. Put images inside the 'images' folder
3. Run:
   python gif_creator.py

#  Output
An animated GIF is generated as 'my_gif.gif'.
